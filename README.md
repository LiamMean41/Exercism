# Exercism solutions

My personal solutions to [Exercism](https://exercism.org) coding exercises, organised by language track.

## Tracks

### TypeScript

| Exercise | Description |
|---|---|
| `hello-world` | Return a "Hello, World!" greeting string |
| `two-fer` | Generate a "one for X, one for me" string with a default parameter |
| `resistor-color` | Map a resistor colour band name to its numeric index |
| `resistor-color-duo` | Decode a two-band resistor value from colour names |

## Structure

Solutions are stored under `solutions/<language>/<exercise-name>/1/`.

## About Exercism

[Exercism](https://exercism.org) is a free, open-source platform offering coding exercises across 70+ language tracks, with a focus on practising and improving through real problems.
